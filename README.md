# Game Portal

## Description
Game Portal is a simple cloud-hosted gaming website developed as part of an academic practical activity.  
The website is deployed using GitHub Pages and provides access to an online game that can be played without login.

## Features
- Home page hosted on GitHub Pages
- Online game accessible through the website
- Repository shared for collaboration
- Guest access to the game

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages (Cloud Hosting)

3. Click on "Play Game" to start the game
